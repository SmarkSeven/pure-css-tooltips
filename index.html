Pure CSS Tooltips

Inside span elements

A lot of web sites use JavaScript to create tooltip but is actually easier doing that with CSS. This is a span element inside a paragraph. You can hover it to look at the tooltip. Use data-tooltip where you want in your HTML code.To show the content put content property equal to the attr() inside the pesudo element and style as you like.

Inside a link as well

Insert the data-tooltip inside the link and use the same style used for the span or other HTML elements.
